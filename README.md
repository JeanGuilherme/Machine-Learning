### Construindo um modelo simples

O conjunto de dados Iris √© amplamente utilizado em Machine Learning como exemplo introdut√≥rio por sua simplicidade e representatividade. Este conjunto cont√©m 150 amostras divididas igualmente entre tr√™s esp√©cies de flores, cada uma descrita por quatro caracter√≠sticas num√©ricas. Sua pequena escala e estrutura clara o tornam ideal para explorar conceitos b√°sicos de classifica√ß√£o.

Neste exemplo, utilizamos o algoritmo Random Forest para criar um modelo que classifica as flores com base nessas caracter√≠sticas. O Random Forest √© uma escolha popular porque combina a robustez de m√∫ltiplas √°rvores de decis√£o, reduzindo a chance de overfitting e produzindo previs√µes confi√°veis. ```python

#### Explica√ß√£o detalhada do c√≥digo

1. **Divis√£o dos dados**: Dividimos o conjunto de dados em duas partes: uma para treino (80%) e outra para teste (20%). Isso permite treinar o modelo em um subconjunto e avaliar seu desempenho em dados desconhecidos, garantindo generaliza√ß√£o.

4. **Fazer previs√µes**: Usamos o modelo treinado para prever os r√≥tulos do conjunto de teste. Essa √© a parte onde o modelo tenta "adivinhar" os resultados com base no que aprendeu.

5. **Avalia√ß√£o do desempenho**: A acur√°cia √© calculada comparando as previs√µes com os valores reais. Ela √© uma m√©trica que indica o percentual de respostas corretas.

#### Etapa 2: Prepara√ß√£o dos Dados
Para este exemplo, criaremos um conjunto de dados fict√≠cio com informa√ß√µes sobre vendas di√°rias de um produto ao longo de um m√™s. Esses dados incluir√£o vari√°veis como o n√∫mero de promo√ß√µes e a quantidade de visitantes na loja.

```python

visitantes = np.random.randint(50, 200, size=30)  # N√∫mero de visitantes di√°rios

    'Dia': dias,
    'Promocoes': promocoes,
    'Visitantes': visitantes,

```

#### Etapa 3: Visualizar os Dados

#### Etapa 4: Construindo o Modelo

```python

from sklearn.metrics import mean_absolute_error

# Sele√ß√£o de vari√°veis

# Divis√£o entre treino e teste

# Fazendo previs√µes
```
